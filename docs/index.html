Smash: About

Smash is an Open Source Shell Mode Javascript Interpreter. Smash features an M185C (Mozilla "SpiderMonkey" 1.8.5 Core) Javascript Interpreter with Dynamic C library interface extensions.

Smash scripts are run in a Unix like environment. Users may run smash scripts with parameters as normal operating system commands. Smash provides builtin-support for normal shell operations such as working with input, output and error handles of the standard C library specification.

Javascript applications written for the smash platform can access all C libraries published on the host system, run system commmands, manage environment variables, write command input, read command output, setup command pipes and other shell like things.
